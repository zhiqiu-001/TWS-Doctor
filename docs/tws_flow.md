# TWS 修复流程

## 功能概述

本工具用于修复 TWS（True Wireless Stereo）蓝牙耳机的各种问题。

## 修复类型

### 1. 扫描单只耳机
扫描并发现附近的 BLE 设备，包括 TWS 耳机的单耳模式。

### 2. 丢失补配/换新单只互联
当丢失一只耳机或更换新耳机时，帮助重新建立左右耳配对。

### 3. 双耳断连不同步修复
修复双耳之间失去同步的问题。

### 4. 一键强制重置耳机底层配对
强制清除耳机的配对信息，回到出厂状态。

### 5. 自动重建左右耳互联通道
自动扫描并重新建立左右耳机之间的连接。

## ESP32-S3 限制

**重要提示：** ESP32-S3 仅支持 BLE（低功耗蓝牙），不支持经典蓝牙。

对于需要经典蓝牙支持的 TWS 耳机，请使用 ESP32-WROOM 芯片。

## 典型修复流程

1. 启动 ESP32 设备
2. PC 上位机连接串口
3. 开始扫描附近的 BLE 设备
4. 选择目标 TWS 耳机
5. 执行相应的修复操作
6. 等待修复完成
